## Feature Summary

Three specialised code-quality agent capabilities:
1. **Simplify Agent** — suggests simpler implementations during review; developer follows TDD and implements only if no tests break.
2. **Secure Coding Agent** — reviews code against secure coding paradigms.
3. **Secret Detection Agent** — scans for secrets/credentials that should not be committed.
