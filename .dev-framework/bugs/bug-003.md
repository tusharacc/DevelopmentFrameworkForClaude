# BUG-003: PO approval artifact missing failed test list from executor

**Severity**: low
**Status**: open
**Created**: 2026-04-11T00:00:00Z
**Source**: Code review of BUG-001 fix (R7)

## Description
The `po-approval` artifact template only includes PO Decision (Approved/Rejected) and Notes. When the PO rejects, there is no structured field listing which specific executor test failures caused the rejection, making it harder for the developer to know what to fix.

## Expected Behavior
The `po-approval` artifact should include a "Failed Tests" section populated from the executor artifact when the decision is Rejected.

## Actual Behavior
No failed test list field exists in the po-approval template.
