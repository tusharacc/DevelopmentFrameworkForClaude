# BUG-016 — pre-commit hook: build_allowlist called per-line (O(n²) file reads)

**Severity:** low
**Status:** open
**Workspace:** code-quality-agents
**Created:** 2026-05-02

## Description

In `skills/code-quality/hooks/pre-commit.sh`, `is_allowlisted()` calls `build_allowlist()` on every invocation. `build_allowlist` rereads and parses `.code-quality-ignore` from disk each time. Since `is_allowlisted` is called for every added line in the diff, this results in O(n) file reads where n is the number of added lines. On a large diff (500+ lines), `.code-quality-ignore` is read 500+ times.

## Fix

Cache the allowlist into an indexed array once before the main loop:
```bash
mapfile -t ALLOWLIST < <(build_allowlist)
```
Then replace the `is_allowlisted` function with an inline array lookup.
