# BUG-001: Review and Test not triggered after coding step

**Severity**: critical
**Status**: open
**Created**: 2026-04-11T00:00:00Z

## Description
When a developer agent finishes its work, the next step should be review followed by testing. However, the framework is currently triggering deploy instead of progressing to the review phase.

## Steps to Reproduce
1. Start a feature or bugfix workflow
2. Complete the development phase
3. Run `/dev hand-off`
4. Observe: deploy is triggered instead of review

## Expected Behavior
After development is complete, `/dev hand-off` should advance to the **Reviewer** phase, then to the **Tester** phase.

## Actual Behavior
The framework skips review/test and triggers deploy after development.
